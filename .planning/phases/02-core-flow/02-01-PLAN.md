<task type="auto">
  <name>Task 1: Create AuthContext.tsx</name>
  <files>src/renderer/context/AuthContext.tsx</files>
  <action>
Create `src/renderer/context/AuthContext.tsx` implementing:

```typescript
// Auth state interface
interface User {
  userId: string;
  username: string;
}

interface AuthState {
  user: User | null;
  isLoading: boolean;
  isAuthenticated: boolean;
}

interface AuthContextValue extends AuthState {
  login: (phone: string, code: string) => Promise<void>;
  register: (phone: string, code: string, username: string) => Promise<void>;
  logout: () => Promise<void>;
  sendSmsCode: (phone: string, type: 'login' | 'register') => Promise<void>;
}
```

**Implementation requirements:**
1. Initialize `TauriAuthClient` with `DiskTokenStorage` and `authServerUrl` from `loadAppConfig()`
2. On mount: Check `AppConfig.auth?.accessToken` exists, call `authClient.validateToken()`, set `isAuthenticated: true` if valid
3. If no token or validation fails: set `isAuthenticated: false`
4. Set `isLoading: true` during initial auth check, set `isLoading: false` after
5. `login()`: call `authClient.smsLogin(phone, code)`, update state to authenticated
6. `register()`: call `authClient.smsRegister(phone, code, username)`, update state
7. `logout()`: call `authClient.logout()`, set state to `user: null, isAuthenticated: false`, dispatch `window.dispatchEvent(new CustomEvent('auth:logout'))`
8. `sendSmsCode()`: call `authClient.sendSmsCode(phone, type)`
9. Wrap with `createContext` and export `AuthProvider` + `useAuth()` hook

**Error handling:**
- Catch errors in login/register/logout, show toast with error message, re-throw
- Map AuthError via `getErrorMessage()` helper (statusCode 400/401/429/500 → Chinese messages)

**Multi-tab sync:**
- Listen for `auth:logout` event on mount, re-check auth state when received
- This ensures all tabs update when any tab logs out
</action>

```typescript
interface OtpInputProps {
  length?: number;  // default 6

  onChange: (value: string) => void;
  error?: boolean;
  disabled?: boolean;
  autoFocus?: boolean;
}
```

**Implementation:**
1. Use 6 individual input boxes (single digit each)
2. Track values as `string[]` array
3. `handleChange(index, value)`: Only allow digits (`/^\d*$/`), auto-focus next input when digit entered
4. `handlePaste(e)`: Prevent default, parse clipboard string (first 6 digits only), fill all 6 inputs, call onChange with joined string
5. `handleKeyDown(e, index)`: On Backspace with empty value, focus previous input
6. Each input: `w-12 h-14 text-center text-2xl font-semibold border rounded-lg`
7. Filled state: `bg-[var(--paper-inset)] border-[var(--line-strong)]`
8. Error state: `border-[var(--error)] focus:ring-[var(--error)]`
9. Focus state: `focus:outline-none focus:ring-2 focus:ring-[var(--accent)] focus:border-[var(--accent)]`
10. Auto-focus first empty input on mount if `autoFocus` is true

**CSS classes per UI-SPEC:**
- Container: `flex gap-2`
- Input box: `w-12 h-14 text-center text-2xl font-semibold border border-[var(--line)] rounded-[var(--radius-sm)] focus:outline-none focus:ring-2 focus:ring-[var(--accent)] focus:border-[var(--accent)] bg-[var(--paper)] transition-all`
- Filled state: `bg-[var(--paper-inset)] border-[var(--line-strong)]`
- Error state: `border-[var(--error)] focus:ring-[var(--error)]`

**Accessibility:**
