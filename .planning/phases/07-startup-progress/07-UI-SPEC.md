## Component Structure

**File:** `src/renderer/components/StartupProgressOverlay.tsx`

**Props:**
```typescript
interface StartupProgressOverlayProps {
  visible: boolean;
  onComplete?: () => void;
}
```

**Internal State:**
```typescript
interface StartupStage {
  stage: 1 | 2 | 3 | 4;
  name: string;      // English
  nameZh: string;     // Chinese
  status: 'pending' | 'active' | 'complete';
}

// Initial state: all pending
const INITIAL_STAGES: StartupStage[] = [
  { stage: 1, name: 'System Core', nameZh: '系统核心', status: 'pending' },
  { stage: 2, name: 'Tray & Management API', nameZh: '托盘与管理 API', status: 'pending' },
  { stage: 3, name: 'Scheduler & Monitors', nameZh: '调度器与监控', status: 'pending' },
  { stage: 4, name: 'Sidecar Ready', nameZh: '助手就绪', status: 'pending' },
];
```

**Event Listening:**
- Listen on `startup:stage` event from `@tauri-apps/api/event`
- Payload: `{ stage: number, name: string, status: 'active' | 'complete' }`
- On `startup:complete` event (no payload): auto-dismiss via `onComplete`

**Timeout:**
- Default: 15000ms (read from `STARTUP_PROGRESS_TIMEOUT_MS` env var if available)
- Falls back to 15000ms if not set

**Step Icons (by status):**
| Status | Icon | Color |
|--------|------|-------|
| pending | Circle (div) | border: --line |
| active | Loader2 (animate-spin) | --accent-warm |
| complete | Check | --success |

## Animation

**Indeterminate Progress Bar:**
```css
/* Track */
.bg-\[var\(--paper-inset\)\] { background: #e8dccf; }
.rounded-full { border-radius: 9999px; }

/* Sliding bar - slides left to right infinitely */
.bg-\[var\(--accent-warm\)\] { background: #c26d3a; }
@keyframes slide {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(400%); }
}
/* Bar width: 30% of track, animation: slide 1.5s ease-in-out infinite */
```

**Spinner:**
```tsx
<Loader2 className="h-10 w-10 animate-spin text-[var(--accent-warm)]" />
```
