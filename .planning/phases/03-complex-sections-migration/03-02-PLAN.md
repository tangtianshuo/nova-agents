- Descriptions/hints: `text-xs text-[var(--ink-muted)]`
- Section titles: `text-sm font-medium text-[var(--ink-muted)]`
- Footer buttons: "取消" (secondary) / "保存" (primary)

# Key behaviors to preserve:

1. **Toggle interaction** — Click toggle → if disabling, immediately toggle off; if enabling, show loading state
2. **Needs config warning** — Show "⚠️ 需要配置 API Key" when server requires config but not provided
3. **Builtin vs custom** — Different edit behavior (handleEditBuiltinMcp vs handleEditMcp)
4. **Enable loading state** — Show Loader2 spinner when enabling/disabling
5. **Discovery links** — Render MCP_DISCOVERY_LINKS with ExternalLink component
