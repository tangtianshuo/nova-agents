**Wave:** 1
**Depends on:** None (parallel-ready)

## Goal

Redesign the native overlay HTML with warm brand aesthetics, staggered CSS animations, light/dark mode support. Keep 4-stage progress structure from Phase 07 but rendered with enhanced animations. Show brand logo + slogan "一念既起，诸事皆成" + version v0.2.2.

---

## Context from Research

- Current overlay is minimal spinner in `lib.rs` line 276
- Overlay uses data URL (no external resources, all inline)
- Dark mode via CSS `prefers-color-scheme` media query (overlay WebView is independent from React)
- Keep design simple to avoid data URL size issues
- Use `transform` and `opacity` only for animations (composited properties, GPU-accelerated)

## Design Specification

### Light Mode
| Element | Value |
|---------|-------|
| Background | `#faf6ee` (--paper) |
| Brand name | `#1c1612` (--ink), 36px, font-weight 300 |
| Slogan | `#6f6156` (--ink-muted), 14px, letter-spacing 0.08em |
| Version | `#a69a90` (--ink-subtle), 11px |
| Progress track | `#e8dccf` (--paper-inset) |
| Progress bar/spinner | `#c26d3a` (--accent-warm) |
| Stage inactive | `#e8dccf` |
| Stage complete | `#c26d3a` |

### Dark Mode (prefers-color-scheme: dark)
| Element | Value |
|---------|-------|
| Background | `#1c1612` (--ink) |
| Brand name | `#faf6ee` (--paper) |
| Slogan | `#a69a90` (--ink-subtle) |
| Version | `#6f6156` (--ink-muted) |
| Progress track | `#3a3532` |
| Progress bar/spinner | `#c26d3a` |
| Stage inactive | `#3a3532` |
| Stage complete | `#c26d3a` |

### Animation Timing
| Animation | Duration | Easing | Delay |
|-----------|----------|--------|-------|
| Brand fade-in | 0.8s | ease-out | 0ms |
| Slogan fade-in | 0.8s | ease-out | 150ms |
| Version fade-in | 0.8s | ease-out | 250ms |
| Progress track fade-in | 0.8s | ease-out | 350ms |
| Spinner fade-in | 0.8s | ease-out | 500ms |
| Stage stagger | 0.4s | ease-out | 0.1s each |
